# Albion Market — Programa Java (JavaFX)

Consulta de preços em tempo real do mercado de Albion Online.
API utilizada: `west.albion-online-data.com` (servidor Americano)


---


## Funcionalidades

- **Filtro por categoria → subcategoria → item** com hierarquia completa
- **Busca por texto** com sugestão automática e tolerância a acentos
- **Tier T1–T8** (ou todos de uma vez)
- **Encantamento @0–@4** (ou todos)
- **Qualidade** Normal a Obra-prima (ou todos — mostra o mais barato independente da qualidade)
- **Seleção de cidades** (8 cidades, sem Avalon)
- **Tabela de resultados** ordenada por tier → encantamento → cidade
- Preços de **venda** (vermelho) e **compra** (verde)
- Tempo relativo de atualização (ex: "3min", "2h", "1d")
